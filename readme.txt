Create a configureable Search-Widget to search politicians on http://www.polixea-profile.de

== Description ==

You can integrate the searchbox on http://www.polixea-profile.de/politiker into your blog sidebar. 
All fields can filled by default, and you can deactivate the parts of the formular.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Design-Tag, Widgets and put Polixea Profile Searchbox to your Sidebar
5. Configure the Searchbox

== Frequently Asked Questions ==

= Can I use it without Widgets? =
 
No, this Plugin integrates a Widget. Without Widgets you can't see this Searchbox

= Can I deactivate the Polixea Profile Banner? =

Yes, go to Wordpress Admin, Design-Tab and configure Polixea-Profile Searchbox

= Does Polixea Profile Searchbox need any Javascript Libary like Protptype? =

No, Polixea Profile nedd no Javascript Libary
